# CLAUDE.md — gstack Edition

## Project: SmartQueue Hospital
A diagnostic-workflow platform built on Garry Tan's recommended stack: FastAPI (Python) + Next.js.

## The gstack Workflow
Follow these phases strictly for every feature:
1. **Think**: Run `/office-hours` (Brainstorming/Forcing Questions).
2. **Plan**: Run `/autoplan` (CEO/Design/Eng review).
3. **Build**: Technical implementation with atomic commits.
4. **Review**: Run `/review` (Staff Engineer audit).
5. **Test**: Run `/qa` (Playwright-based browser testing).
6. **Ship**: Run `/ship` (Documentation update + Git push).

## Technical Principles
- **API First**: The Python FastAPI backend is the source of truth.
- **Agentic Decisioning**: We use reasoning loops for queue optimization.
- **Premium UX**: Next.js with high-end Vanilla CSS aesthetics.

Available Skills:
